Golden Crave 🍽️ 
A Full-Stack Restaurant Management System with Online Payments & Reservation Features - built with PHP, MySQL, JavaScript, and Bootstrap.

Golden Crave allows customers to browse the restaurant menu, place orders, reserve dining tables, and complete secure online payments using PayPal and credit/debit cards.

🚀 Features
Dynamic menu connected to MySQL database
Customer authentication system
Food ordering system
Table reservation system
Date & time selection
Number of guests
Reservation history linked to each user
Checkout & online payment integration
PayPal
Credit/Debit Cards
CRUD operations for:
Orders
Reservations
Admin Dashboard
Manage orders
Update order status
View reservations
Control customer activities
Responsive UI using Bootstrap

🛠️ Technologies Used
HTML5
CSS3
Bootstrap
JavaScript
PHP
PDO
MySQL
